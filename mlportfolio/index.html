NLP: A Speech Detection

Developed a hate speech detection system using NLP, which involved preprocessing using Regex, NLTK, Gensim, and SpaCy. The words were tokenized using BoW, TF-IDF, and GloVe, and the model was fine-tuned using LSTM, Bidirectional LSTM, and BERT.
